# Average

The purpose of this program is to find the average value of input integer values. When the user inputs 0, program automatically stops and starts calculating the average value(not including zero).

### Examples

---
Input: `1 2 3 4 5 0`

Output: `3`

---
Input: `1 2 3 4 6 0`

Output: `3`

---
Input: `1 2 3 4 0 `

Output: `2`

---
Input: `1 1 9 0`

Output: `3`
